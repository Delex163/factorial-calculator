# Факториал Калькулятор

Простая программа на Python для вычисления факториала положительного целого числа с оптимизацией для больших чисел.

## Возможности

- Ввод положительного целого числа с клавиатуры
- Обработка ошибок (нечисловой ввод, отрицательные числа, пустой ввод)
- Оптимизированное вычисление через `math.factorial`
- Поддержка очень больших чисел (до 10000+)
- Информативный вывод результата

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/Delex163/factorial-calculator.git
cd factorial-calculator

# Запуск (требуется Python 3.6+)
python factorial.py
